# Technical Stack

## Frontend
- **Framework**: Next.js 16 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript
- **PDF Viewing**: @document-kits/viewer, @embedpdf/react-pdf-viewer, react-pdf, pdfjs-dist

## Backend
- **Runtime**: Node.js
- **Framework**: Express 5
- **Database**: PostgreSQL (pg)
- **File Storage**: AWS S3
- **Payment Gateway**: Razorpay
- **Authentication**: JWT, bcryptjs
- **PDF Generation**: pdfkit

## Infrastructure
- **Hosting**: Likely Vercel (vercel.json present)
- **Deployment**: Git-based
